# Assets Folder

This folder contains static assets (images, logos, etc.) used by the application.

## Terrace Finance Logo

Place the `terrace-finance-logo.png` file in this directory.

The logo should be:
- Format: PNG (with transparency preferred)
- Recommended size: 80-120px width (height will auto-scale)
- File name: `terrace-finance-logo.png`

## Usage

Assets in this folder are served at `/assets/` URL path.

Example:
- File: `public/assets/terrace-finance-logo.png`
- URL: `/assets/terrace-finance-logo.png`
